Features

A cadboxx is a 3D Workspace and Physics Environment for creating dynamic joints between 3D Models that form an assembly of moving parts. All scenes can be used by multiple controllers and any device, but the commands available will differ depending on your hardware. Import files from any 3D system and combine them with your own custom parts.

Our Expertise

We make building mechanical assemblies easier by providing a library of parametric 3D Models in GLB Format. All Models are editable and interactive when placed into a Physics scene.

Import Models & Set-Up

Importing your models could not be any easier, just create a Spoke account (which is also your Hubs account) and select the "My Assets" tab from the Spoke menu. Now create your own private library of parts for one-click deployment into any future apps or scenes. If you have a static 3D Assembly, then there is an option to upload the assembly in one go.

The System

Stand-alone

Self-contained

The simplest way to get going with cadboxx is to use it as a stand-alone system. No set-up is required and the user can just open up the web app and start using thw cadboxx to manipulate their models. Build new geometry components and apply textures from our own library of metals and finishes.

Hubs link

By far the most effective way to use cadboxx is to exploit the advantages offered by the links between GitHub, Hubs by Mozilla and Sketchfab. Build your own custom library in Spoke and import all the models into cadboxx as an assembly in a single file. Then just use the tools in cadboxx to apply the constraints and set-up the scene.
