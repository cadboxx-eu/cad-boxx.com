WHY CAD-BOXX.COM?

We make building mechanical assemblies easier by providing a library of parametric 3D Models which can be placed into your scene in a single click. All Models are editable and interactive when placed into a scene which can be Physics-based or not.
The service offers a unique blend of open source and specialist code writen in-house. All OS sections are free!
Upload a Model to our Cloud storage to build your own private asset library based in the Cloud. Build scenes by dropping these models into a new scene created with SketchFab models. Add your own custom parts, and then define how they all interact by setting the names of the components in the scene.
That´s right, just edit the name of each part to define how they move.

The System

Stand-alone

Self-contained

The simplest way to get going with cadboxx is to use it as a stand-alone system. No set-up is required and the user can just open up the web app and start using thw cadboxx to manipulate their models. Build new geometry components and apply textures from our own library of metals and finishes.

Hubs link

By far the most effective way to use cadboxx is to exploit the advantages offered by the links between GitHub, Hubs by Mozilla and Sketchfab. Build your own custom library in Spoke and import all the models into cadboxx as an assembly in a single file. Then just use the tools in cadboxx to apply the constraints and set-up the scene.
